# Mod the Cube - Unity Scripting Challenge

## Overview
This project is my solution to the "Mod the Cube" challenge from Unity Learn. The goal was to take a basic Unity scene and use C# scripting to dynamically alter the behavior and appearance of a GameObject at runtime.

## Features
I programmed the cube to update its properties automatically every 2 seconds. The changes include:
* Random Colors: Changing to new RGBA values.
* Random Size Modification: Scaling on different axes.
* Random Rotation: Changing rotation speeds and the angles of rotation.

## What I Learned
* This challenge allowed me to utilize what I have previously learned, strengthening my grasp on concepts, especially on invoking methods.
